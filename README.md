# Selenium Python Automation Framework

This is a demo automation project built using:

- Python
- Selenium WebDriver
- PyTest
- Page Object Model (POM)

## Project Structure


SeleniumProject/
│
├── tests/
├── pages/
├── drivers/
├── conftest.py
├── requirements.txt
└── README.md


## Setup Instructions

1. Install Python
2. Install dependencies


pip install -r requirements.txt


3. Run tests


pytest -v


4. Generate HTML report


pytest --html=report.html


## Test Scenarios

- Valid Login Test
- Invalid Login Test
